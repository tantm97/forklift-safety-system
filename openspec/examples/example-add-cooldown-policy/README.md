# example-add-cooldown-policy (read-only worked example)

Lite-workflow example for `forklift-safety-system`. Demonstrates a small,
contained C++ feature: a configurable per-(camera, forklift_track) cooldown
to de-duplicate `person_near_forklift` alerts.

Files:

- `meta.yaml` — change metadata
- `brief.md`  — what / why / scope / acceptance criteria
- `tasks.md`  — leaf-actionable checklist
